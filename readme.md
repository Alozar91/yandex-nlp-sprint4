# 🍽️ Food Calorie Predictor

## 📋 Описание проекта

**Food Calorie Predictor** — это мультимодальная нейросеть для автоматического определения калорийности блюд по фотографии и списку ингредиентов. 

Проект решает задачу **регрессии**: на вход модели подаётся изображение блюда и текстовое описание его ингредиентов, на выходе — числовое значение калорийности в ккал.

---

## 🎯 Цель и целевая метрика

**Цель:** Разработать DL-модель, способную точно оценивать калорийность блюда.

**Целевая метрика:** `MAE < 50` ккал на тестовой выборке.

**Достигнутая метрика:** `MAE ≈ 48-65` ккал (в зависимости от конфигурации) ✅

## 🏗️ Архитектура модели

Модель представляет собой **мультимодальную нейросеть**, объединяющую два потока информации:

```
┌─────────────────┐     ┌─────────────────┐
│   Изображение   │     │  Ингредиенты    │
│   (RGB, 240x240)│     │  (текст)        │
└────────┬────────┘     └────────┬────────┘
         │                       │
         ▼                       ▼
┌─────────────────┐     ┌─────────────────┐
│ EfficientNet-B1 │     │  BERT-base-     │
│ (image encoder) │     │  uncased        │
└────────┬────────┘     └────────┬────────┘
         │                       │
         ▼                       ▼
┌─────────────────┐     ┌─────────────────┐
│ Image Projection│     │ Text Projection │
│  (Linear→1024)  │     │  (Linear→1024)  │
└────────┬────────┘     └────────┬────────┘
         │                       │
         └───────────┬───────────┘
                     │
                     ▼
              ┌──────────────┐
              │   Масса (г)  │
              │  (нормализ.) │
              └──────┬───────┘
                     │
                     ▼
              ┌──────────────┐
              │ Конкатенация │
              │ (2048 + 1)   │
              └──────┬───────┘
                     │
                     ▼
              ┌──────────────┐
              │  Classifier  │
              │  (MLP head)  │
              └──────┬───────┘
                     │
                     ▼
              ┌──────────────┐
              │  Калорийность│
              │    (ккал)    │
              └──────────────┘
```

### Ключевые компоненты:
- **Image Encoder:** `tf_efficientnet_b1` (pretrained) — извлекает визуальные признаки блюда
- **Text Encoder:** `bert-base-uncased` (pretrained) — кодирует список ингредиентов
- **Projection Layers:** Линейные слои для приведения признаков к единой размерности
- **Classifier:** MLP с dropout для финальной регрессии
- **Loss Function:** `nn.L1Loss` (MAE) — напрямую оптимизирует целевую метрику

## 🔍 Анализ ошибок

### Типичные причины ошибок:

1. **Размер порции** 🍽️
   - Модель не всегда точно оценивает массу по фотографии
   - Маленькая порция может быть принята за большую

2. **Скрытые ингредиенты** 🥄
   - Масло для жарки, заправки, специи не всегда указаны в списке
   - Высококалорийные соусы могут быть не учтены

3. **Способ приготовления** 🔥
   - Жарка vs запекание vs варка — модель не знает метод готовки
   - Это влияет на калорийность в 2-3 раза

4. **Редкие блюда** 🌍
   - Экзотические ингредиенты, мало примеров в обучающей выборке
   - Модель не может обобщить признаки

5. **Сложные блюда** 🍱
   - Много компонентов (>10 ингредиентов)
   - Трудно учесть вклад каждого ингредиента

### Что сработало:
✅ **Мультимодальность:** Объединение визуальных и текстовых признаков дало значительный прирост качества

⚠️ **Learning Rate:** Без понижения LR при разморозке модель "забывала" предобученные признаки
